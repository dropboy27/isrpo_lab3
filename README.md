## 1. Название плагина
**Insert FIO** — расширение для Visual Studio Code, которое добавляет пользовательскую команду по вставке заранее заданного ФИО в текущую позицию курсора.

---

## 2. Разработанное расширение

### Назначение
Добавляет собственную команду `Insert: ФИО`, которая вставляет заданное пользователем ФИО в текущую позицию курсора в открытом файле.

### Основные возможности
- Регистрация пользовательской команды в палитре VS Code.  
- Работа с активным текстовым редактором.  
- Вставка текстовых данных в выбранную позицию.
